### v7_graph_nodes.py
**Purpose:** Implements all 5 agents  
**Contains:**
- `Nodes` class with 5 methods (one per agent)
- `retrieve()` - Agent 1: Hybrid retrieval
- `rerank()` - Agent 2: BGE reranking
- `grade_documents()` - Agent 3: Document filtering
- `generate()` - Agent 4: Answer generation
- `transform_query()` - Agent 5: Query rewriting

**Key Functions:**
```python
class Nodes:
    def retrieve(state) -> retrieved_docs
    def rerank(state) -> reranked_docs
    def grade_documents(state) -> filtered_docs
    def generate(state) -> final_answer
    def transform_query(state) -> new_query
```

**Dependencies:**
- `k8s_hybrid_retriever.py` (retrieval)
- `v7_bge_reranker.py` (reranking)
- `llama_stack_client` (LLM calls)

**When to modify:**
- Change agent behavior
- Update prompts
- Adjust grading criteria
- Modify output format

## üéØ Quick Reference: "I need to..."

### Change chunk size
‚Üí Edit `k8s_hybrid_retriever.py`, line ~128  
‚Üí Change `chunk_size=1000` to your value

### Modify agent prompts
‚Üí Edit `v7_graph_nodes.py`  
‚Üí Find the agent method (retrieve, grade, generate, etc.)  
‚Üí Update prompt strings

### Add new environment variable
‚Üí Edit `v7-deployment.yaml`  
‚Üí Add to `env:` section  
‚Üí Redeploy: `oc apply -f v7-deployment.yaml`

### Change LLM model
‚Üí Edit `v7-deployment.yaml`  
‚Üí Change `LLAMA_STACK_MODEL` value  
‚Üí Restart pod

### Update code
‚Üí Edit Python files  
‚Üí Run: `oc create configmap ... --dry-run=client -o yaml | oc apply -f -`  
‚Üí Restart: `oc rollout restart deployment/smart-logging`

### Debug agents
‚Üí Check logs: `oc logs deployment/smart-logging`  
‚Üí Look for: "NODE 1", "NODE 2", etc.  
‚Üí Check: "Retrieved X documents", "Filtered to Y documents"

### Fix detection issues
‚Üí Read `CRITICAL_FIXES_APPLIED.md` first  
‚Üí Check chunk size (should be 1000)  
‚Üí Verify full document grading (no [:500])  
‚Üí Confirm inclusive philosophy in prompts
